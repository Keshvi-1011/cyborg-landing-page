# 🤖 Cyborg AI Landing Page

A futuristic cyberpunk-inspired landing page built with modern web technologies, featuring immersive animations, premium UI/UX, holographic effects, and AI-inspired design elements.

## 🚀 Features

* Futuristic Cyborg / Cyberpunk Design
* Fully Responsive (Desktop, Tablet & Mobile)
* Glassmorphism & Neon UI Effects
* Interactive Holographic Dashboard
* Scroll-Based Animations
* Floating Particles & Neural Network Backgrounds
* Animated Feature Cards
* TechFest & AI Showcase Sections
* Modern Motion Design

## 🛠️ Tech Stack

* React.js / Next.js
* Tailwind CSS
* Framer Motion
* GSAP
* React Icons

## 🎯 Inspiration

Inspired by futuristic AI systems, cybernetic interfaces, sci-fi technology, and cyberpunk worlds such as Blade Runner, Tron, Ghost in the Shell, and Cyberpunk 2077.
